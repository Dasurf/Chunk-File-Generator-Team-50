Here are some common questions about ChunkFiles

How are files split?
  • Step 1: Select the file format that you want eg. (.CSV, .JSON, e.tc)
  • Step 2: Click on the “Upload File” button and select the desired file from your device or fill in the web link to where the file is located in the input box provided
  • Step 3: Select the number of rows that you want each file to have (Optional)
  • Step 4: Click on the “Split File” button and wait for the operation to finish and display the various files.
  • Step 5: Click on the “Download” button and save the zip file on your device.

The file size limit is 1GB
